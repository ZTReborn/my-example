#单例模式
1、单例类只能有一个实例。
2、单例类必须自己创建自己的唯一实例。
3、单例类必须给所有其他对象提供这一实例。

实现单例模式有饿汉式和懒汉式和静态内部类

##饿汉式
###优点：
代码实现简单
利用类加载机制避免了多线程同步问题

###缺点：

在类加载时就完成了实例化，没有达到Lazy loading的效果，有可能造成内存浪费    


##懒汉式

起到了Lazy loading效果，适合在单线程环境中使用
添加synchronized双重锁解决多线程问题仍会出现发射问题
最后使用三重锁解决


##静态内部类
利用JVM加载静态内部类的机制保证多线程安全
实现Lazy loading效果
效率高
